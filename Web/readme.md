- Cold: 열선패드의 온도가 13도 이상이 되면 전원이 꺼지게 설정되는 버튼입니다.
- Warm: 열선패드의 온도가 16도 이상이 되면 전원이 꺼지게 설정되는 버튼입니다.
- Hot: 열선패드의 온도가 20도 이상이 되면 전원이 꺼지게 설정되는 버튼입니다.
